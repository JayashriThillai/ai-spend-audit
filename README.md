# AI Spend Audit

AI Spend Audit is a web application that helps startups identify overspending on AI tools like ChatGPT, Claude, Cursor, and GitHub Copilot.

Users can enter their AI tool usage, monthly spend, and team size to receive personalized cost-saving recommendations and potential monthly savings.

## Features

- AI tool spend audit
- Personalized savings recommendations
- Dynamic audit results
- Persistent form data using localStorage
- Responsive UI with Tailwind CSS

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
